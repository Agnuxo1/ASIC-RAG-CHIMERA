# chimera-handoff (Python)

The Python subtree contains runnable CLIs and tests.

## CLIs

After running `../scripts/setup_python.sh`:

- `chimera-pc-chronos-sweep` — generate PC‑CHRONOS run roots (Chronos-only by default).
- `chimera-summarize` — write `SUMMARY.csv` + `REPORT.md` from a run root.
- `chimera-did` — compute seed-paired DiD + `DECISION: PASS/FAIL`.
- `chimera-verify` — validate a run root against the schema contract.
